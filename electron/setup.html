Install Tesseract OCR? (optional)

For reading text in scanned PDFs and images, the app uses Tesseract (~50 MB). On Windows, install uses winget and may prompt for Microsoft Store source terms.

You can skip and add OCR later in Settings; sorting still works, with weaker results on scans.
